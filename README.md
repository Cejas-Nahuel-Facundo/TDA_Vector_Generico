# TDA_Vector_Generico
Implementacion de un Tipo de Dato Abstracto(TDA) generico en C para distintas operaciones.
## Caracteristicas
- Vector Dinamico Generico(void*).
- Insercion simple y ordenada.
- Comparadores Genericos.
- Funciones de Visualizacion.
